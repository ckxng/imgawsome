Imgawsome

AWS-backed image upload service

View project on GitHub

Architecture

  1. Elastic Beanstalk - Node.js Application Hosting
  2. DynamoDB - User database
  3. S3 - Image storage

Authors

This is a personal, fun-project for Ian and Cameron to experiment with JavaScript, HTML5, Node.js, and AWS.
